⚙️ চ্যানেল অ্যাডমিন কন্ট্রোল প্যানেল

**গুরুত্বপূর্ণ নোট:** এটি একটি সার্ভারবিহীন প্যানেল। চ্যানেলগুলি আপডেট করতে আপনাকে সরাসরি GitHub-এর ফাইল এডিটিং ইন্টারফেস ব্যবহার করতে হবে।

১. চ্যানেল ডেটা এডিট করুন (যোগ/এডিট/ডিলিট)

চ্যানেলগুলির তালিকা পরিবর্তন (যেমন, নতুন চ্যানেল যোগ বা কোনো চ্যানেলের URL পরিবর্তন) করতে নিচের বাটনে ক্লিক করুন।

আপনাকে JSON ফরম্যাটটি সঠিকভাবে অনুসরণ করতে হবে, যেখানে প্রতিটি চ্যানেলের জন্য `name`, `logo` এবং `url` অবশ্যই থাকতে হবে।

  • **`name`**: চ্যানেলের নাম।
  • **`logo`**: লোগোর URL (https)।
  • **`url`**: চ্যানেলের স্ট্রিমিং URL (M3U8/HLS)।
channels_data.json এডিট করুন

**সেভ করার নিয়ম:** GitHub-এ এডিট করার পর অবশ্যই "Commit changes" (বা "Save changes") বাটনে ক্লিক করবেন।

২. লাইভ প্লেয়ার দেখুন

পরিবর্তনগুলি সেভ করার পর লাইভ প্লেয়ার কেমন দেখাচ্ছে, তা যাচাই করুন।
